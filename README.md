#  Password Generator

A secure random password generator built with Python.

##  About The Project
Generates strong, secure passwords based on your preferences — choose length, include/exclude uppercase, lowercase, numbers and symbols.

##  Features
- Custom password length
- Toggle uppercase, lowercase, numbers, symbols
- Password strength checker
- Generate multiple passwords at once
- Copy-ready output

##  Built With
- Python 3
